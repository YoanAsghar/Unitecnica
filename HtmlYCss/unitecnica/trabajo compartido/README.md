# Proyecto Web Grupal - IA Store

## Instrucciones para el Equipo de Desarrollo

Este es un proyecto colaborativo de 7 personas. Cada integrante tiene un componente asignado dentro del archivo `index.html`.

### Flujo de Trabajo
1.  **Clona** el repositorio.
2.  Busca tu sección designada mediante los comentarios en el archivo `index.html` (e.g., `<!-- COMPONENTE X: ... -->`).
3.  Desarrolla tu parte dentro del `div` correspondiente.
4.  **No modifiques** el código de tus compañeros ni el JavaScript de navegación general.
5.  Usa clases CSS únicas para evitar conflictos de estilo.

### Asignaciones
- **Persona 1:** Barra de Navegación (Navbar) y Estructura Base.
- **Persona 2:** Menú de Login (Bloqueo inicial de página).
- **Persona 3:** Catálogo de Productos (Marketplace).
- **Persona 4:** Gestión de Usuarios (Lista y Perfiles).
- **Persona 5:** Dashboard (Gráficos y Estadísticas).
- **Persona 6:** Administración (Botones de CRUD).
- **Persona 7:** Carrito y Pedidos.

### Tecnologías
- HTML5
- CSS3
- JavaScript (Mínimo para navegación)
